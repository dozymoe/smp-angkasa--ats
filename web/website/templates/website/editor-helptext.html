{% trans "Headings" %}

Untuk setiap huruf di dalam judul bab, tuliskan simbol (contohnya, = atau *) dengan jumlah yang sama.

Mengenai simbol tersebut, karakter yang dipakai sebenarnya tidak penting, urutan penggunaan simbol tersebut yang menjadikannya <h1> atau <h2>. Ikuti penggunaan simbol di bawah agar menghasilkan teks yang seragam.

{% trans "Bulleted list" %}

    - This is a bulleted list.
    - It has two items, the second
      item uses two lines.

Hasil:

  • This is a bulleted list.
  • It has two items, the second item uses two lines.

Contoh:

    * This is a bulleted list too.
    * The principle stays the same.

Hasil:

  • This is a bulleted list too.
  • The principle stays the same.

  1. This is a numbered list.

    1. This is a numbered list too.
    2. Use this format to specify the numbering.

  1. This is a numbered list too.
